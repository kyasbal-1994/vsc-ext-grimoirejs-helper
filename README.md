# grimoire-js Helper

## Features

Syntax highlighter for following.
* `GOML` - Geometry Object Markup Language for Grimoire.js
* `SORT` - GLSL extended for Grimoire.js

Snippets for GOML

## Requirements

This extension requires Shader languages support for VS Code.
